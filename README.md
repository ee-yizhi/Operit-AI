📌 简介

「主动关怀」是一个智能关怀系统，通过定时检测用户状态，在合适时机发送关怀消息。系统支持夜间免打扰、随机触发率、屏幕状态检测、前台应用分类识别等功能，旨在为用户提供恰到好处的关怀，避免打扰。


🎯 功能特性


⏰ 定时触发: 每15分钟自动检测一次

🎲 随机触发: 约40%的实际触发率（随机数≥4时触发）

🌙 夜间免打扰: 23:00-07:00期间不触发

💬 对话状态检测: 用户正在对话时不打扰

📱 屏幕状态检测: 手机息屏时不触发

📊 应用分类识别: 智能识别前台应用类型

📩 自动发送关怀: 在合适时机发送温馨提醒


🔄 工作流程

graph TD
    A[定时触发<br>每15分钟] --> B{随机数≥4?}
    B -- 否 --> Z[结束]
    B -- 是 --> C{夜间?<br>23:00-07:00}
    C -- 是 --> Z
    C -- 否 --> D{对话空闲?}
    D -- 否 --> Z
    D -- 是 --> E{屏幕亮起?}
    E -- 否 --> Z
    E -- 是 --> F[获取前台应用]
    F --> G{学习/工作类?}
    G -- 是 --> Z
    G -- 否 --> H{娱乐/休闲类?}
    H -- 是 --> I[发送关怀消息]
    H -- 否 --> Z

📋 应用分类规则

✅ 学习/工作类（不打扰）


不背单词、作业帮、笔记、WPS、Office、超星等

匹配关键词: zuoyebang|beike|buubei|tianxuewang|sxew|note|notion|wps|office|chaoxing|study


🎮 娱乐/休闲类（触发关怀）


抖音、快手、B站、美团、闲鱼、淘宝等

匹配关键词: douyin|aweme|kuaishou|bilibili|jianying|xiangtu|meitu|meituan|xianyu|taobao|game


⚖️ 中性类（不打扰）


微信、QQ、支付宝、相机、相册等

默认分类为中性，不触发关怀


⚙️ 配置说明

定时设置


触发间隔: 15分钟 (900,000ms)

随机触发率: ~40% (随机数1-5, ≥4时触发)


时间设置


夜间免打扰: 23:00 - 07:00


关怀消息


固定消息: "休息一会吧，别太累了。"
